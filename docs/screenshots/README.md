# Screenshots Needed

This directory contains screenshots for the Luno README and documentation.

## Required Screenshots

### 1. Hero Image/GIF (`hero.gif`)
An animated GIF showing:
1. Right-clicking on a word in a webpage
2. Selecting "Quick Define" from the context menu
3. Luno popup opening with the definition
4. Duration: 2-3 seconds
5. Recommended size: 800x600px or similar

**How to Create:**
- Use screen recording software (OBS, ScreenFlow, LICEcap, etc.)
- Record the interaction on a clean webpage
- Convert to GIF using tools like ezgif.com or Gifski

### 2. Definition View (`definition-view.png`)
Show the main popup with:
- Search bar at top
- Definition tab selected
- A word defined (e.g., "serendipity")
- Phonetic pronunciation visible
- Multiple definitions with examples
- Dark theme showcased
- Recommended size: 400x600px (actual popup size)

### 3. History Tab (`history.png`)
Show the history tab with:
- List of previously searched words
- Timestamps visible
- Clear history button visible
- At least 5-10 history entries
- Dark theme
- Recommended size: 400x600px

### 4. Settings Tab (`settings.png`)
Show the settings tab with:
- Theme selector (Auto/Dark/Light)
- Keyboard shortcut display
- About section with version number
- Links to check updates and report issues
- Dark theme
- Recommended size: 400x600px

## Taking Screenshots

### Chrome Extensions
1. Load the Luno extension
2. Click the icon to open the popup
3. Use Chrome DevTools screenshot feature:
   - Right-click popup â†’ Inspect
   - Press Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows/Linux)
   - Type "Capture screenshot"
   - Choose "Capture node screenshot" and select the popup element

### Recommended Tools
- **macOS**: Cmd+Shift+4 (native), CleanShot X
- **Windows**: Snipping Tool, Greenshot
- **Linux**: GNOME Screenshot, Flameshot
- **Cross-platform**: Chrome DevTools

## Image Optimization

After capturing screenshots:
- Optimize PNG files using TinyPNG or ImageOptim
- Keep file sizes under 500KB each
- Use descriptive filenames
- Ensure images show the dark theme clearly
